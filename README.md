# 水滴 QQ（ShuDi）

轻量级 **QQ NT 多账号登录控制台**。本机一键启动，网页管理多个 QQ，实时收消息，并以 OneBot 风格 WebSocket 对外推送，方便对接机器人 / 业务系统。

当前版本：**1.0.3**

---

## 能做什么

- **多账号管理**：扫码 / 密码登录，列表集中管理在线状态与进程
- **消息收发**：群聊、私聊、入退群、管理变更、撤回等事件入库与展示
- **通讯推送**：OneBot v11 风格入站 WS，字段含 `group_id`、`raw_message`、`message` 段数组、`sender`、`group_name` 等
- **开放 API**：本地 HTTP / Bridge API，便于二次开发
- **授权体系**：账号验证、卡密充值、设备绑定、版本推送与停用
- **运营能力**：公告、广告位、维护模式、实时心跳会话

---

## 技术架构

| 模块 | 说明 |
|------|------|
| 控制台前端 | Vue 3 + Element Plus |
| 本地 Host | C++（`dripqq-host`），无 Java 运行时依赖 |
| QQ 桥接 | QQ NT 注入桥（hook-bridge） |
| 验证后台 | 独立管理端（账号 / 卡密 / 版本 / 公告） |

发布包双击 `ShuDi.bat` 即可启动，默认控制台：`http://127.0.0.1:8080`

---

## 快速开始

1. 解压发布包到任意目录  
2. 确认本机已安装 **QQ NT**  
3. 双击 `ShuDi.bat` / `start.bat`  
4. 浏览器打开控制台，注册或登录授权账号后添加 QQ
